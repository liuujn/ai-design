# 购物车管理画面设计

## 画面概要

| 項目 | 説明 |
|------|------|
| 画面名 | 购物车管理 |
| 機能 | 购物车的商品管理、数量修改、地址选择、结算下单 |
| 利用者 | 登录用户 |
| 関連API | GET/POST/PUT/DELETE /api/v1/carts/* |

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 购物车一览 | 显示当前用户的购物车内容，包括商品名、单价、数量、小计 |
| 2 | 数量变更 | 增减商品数量，实时更新小计和总金额 |
| 3 | 删除商品 | 从购物车中删除指定商品 |
| 4 | 地址选择 | 从用户地址列表中选择配送地址 |
| 5 | 清空购物车 | 一键清空所有商品 |
| 6 | 结算下单 | 将购物车转换为订单 |

## 一覧項目

| 列 | 説明 | 表示形式 |
|----|------|---------|
| # | 序号 | 数字 |
| 商品 | 商品名称 | 文本 |
| 单价 | 商品单价 | 金额 ¥xxx.xx |
| 数量 | 购买数量 | 数字 + 增减按钮 |
| 小计 | 数量×单价 | 金额 ¥xxx.xx |
| 操作 | 删除按钮 | 删除链接 |

## 主要操作

### 数量变更
- 点击 `[-]` 减少数量（最小为1）
- 点击 `[+]` 增加数量
- 调用 `PUT /api/v1/carts/items/{id}` 更新数量

### 地址选择
- 点击 `[修改]` 弹出地址选择对话框
- 调用 `GET /api/v1/addresses?userId={currentUserId}` 获取地址列表
- 选择后调用 `PUT /api/v1/carts/address` 保存

### 结算下单
- 校验：购物车不为空 且 已选择地址
- 调用 `POST /api/v1/carts/checkout`
- 成功后跳转到订单详情页
