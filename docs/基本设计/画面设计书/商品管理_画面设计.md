# 商品管理画面

## 1. 基本信息

- 画面ID(路径): /product
- 画面名: 商品管理

## 2. 页面概要

- 本页面用于管理商品信息，支持商品的搜索、新建、编辑、详情查看和删除操作。
- 商品列表显示所有商品，支持按关键词、分类、状态进行搜索。
- 列表展示商品概要信息：名称、价格、库存、分类、状态等。
- 点击编辑按钮可跳转至商品编辑画面。
- 点击新建按钮可跳转至商品创建画面。

## 3. 页面布局

### 3.1 商品列表画面

#### 补充说明

- 上部：操作按钮区域（新建）
- 中部：商品列表DataTable + 分页器
  - 支持按关键词、分类、状态过滤
  - 商品状态使用标签组件显示不同颜色
  - 价格显示带货币符号

### 3.2 商品编辑画面

#### 补充说明

- 上部：页面标题
- 中部：商品信息编辑表单（垂直布局）
  - 商品信息区域：名称、描述、价格、库存
  - 分类下拉选择
  - 图片URL输入
  - 商品状态下拉选择
- 底部：保存/取消/返回按钮
- 创建模式下所有字段均可编辑
- 编辑模式下可修改所有字段
- 查看模式下所有字段为只读

## 5. 处理功能详细

### 5.1 初始显示处理 (initializePage)

- 调用时机: 商品列表页面初次显示时
- 处理内容:
  - 自动加载商品列表
  - 如果没有搜索结果，显示空状态

### 5.2 搜索处理 (handleSearch)

- 调用时机: 搜索按钮点击时
- 处理内容:
  - 保存搜索条件输入值
  - 调用商品列表搜索API获取数据
  - 更新列表显示

### 5.3 清除条件处理 (handleClear)

- 调用时机: 清除条件按钮点击时
- 处理内容:
  - 将搜索条件恢复为初始值
  - 重新加载全部商品列表

### 5.4 新建跳转处理 (handleCreate)

- 调用时机: 新建按钮点击时
- 处理内容:
  - 跳转至编辑画面，空白表单，模式为创建

### 5.5 编辑跳转处理 (handleEdit)

- 调用时机: 点击编辑图标时
- 处理内容:
  - 跳转至商品编辑画面，加载商品详情数据

### 5.6 保存处理 (handleSave)

- 调用时机: 点击保存按钮时
- 处理内容:
  - 校验必填字段：商品名称、价格、库存
  - 校验价格大于0
  - 校验库存大于等于0
  - 构建请求数据，调用商品创建/更新API
  - 创建模式：POST /api/v1/products
  - 编辑模式：PUT /api/v1/products/{productId}（附带updatedAt）
  - 保存成功后返回商品列表页面

### 5.7 删除处理 (handleDelete)

- 调用时机: 点击列表中的删除图标时
- 处理内容:
  - 显示删除确认对话框
  - 确认后调用商品删除API（DELETE /api/v1/products/{productId}）
  - 请求参数：updatedAt（乐观LOCK）
  - 成功后刷新列表

### 5.8 状态变更处理 (handleToggleStatus)

- 调用时机: 点击列表中的上架/下架图标时
- 处理内容:
  - 显示确认对话框
  - 调用商品状态变更API（PATCH /api/v1/products/{productId}/status）
  - 请求参数：status, updatedAt（乐观LOCK）
  - 成功后刷新列表
