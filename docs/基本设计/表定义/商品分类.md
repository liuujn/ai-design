# 商品分类表设计

## 商品分类主数据

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 分类唯一标识 | VARCHAR | 36 | TRUE | | PK |
| name | 分类名称 | VARCHAR | 100 | TRUE | |  |
| description | 分类描述 | VARCHAR | 500 | FALSE | |  |
| sort_order | 排序序号 | NUMBER(5) | | FALSE | 0 | 升序排列 |
| status | 分类状态 | VARCHAR | 20 | TRUE | 'active' | active/inactive |
| created_at | 创建时间 | TIMESTAMP | | TRUE | CURRENT_TIMESTAMP | |
| created_by | 创建者ID | VARCHAR | 36 | TRUE | | |
| updated_at | 更新时间 | TIMESTAMP | | TRUE | CURRENT_TIMESTAMP | |
| updated_by | 更新者ID | VARCHAR | 36 | TRUE | | |
| is_deleted | 逻辑删除标志 | NUMBER(1) | | TRUE | 0 | |

- Table Name: categories
- PK: categories_pk
- INDEX: categories_status_idx (status)
