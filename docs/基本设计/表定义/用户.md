# 用户主数据表设计

## 用户主数据

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 用户唯一标识 | VARCHAR | 36 | TRUE |  | PK |
| username | 用户账号 | VARCHAR | 50 | TRUE |  | UNIQUE |
| display_name | 用户显示名 | VARCHAR | 100 | TRUE |  |  |
| email | 邮箱地址 | VARCHAR | 255 | TRUE |  | UNIQUE |
| phone | 电话号码 | VARCHAR | 20 | FALSE |  |  |
| password_hash | 密码哈希值 | VARCHAR | 255 | TRUE |  |  |
| status | 账户状态 | VARCHAR | 20 | TRUE | 'active' |  |
| mfa_enabled | 是否启用多因素认证 | BOOLEAN |  | TRUE | false |  |
| mfa_secret | MFA密钥 | VARCHAR | 255 | FALSE |  |  |
| created_at | 创建时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| created_by | 创建者ID | VARCHAR | 36 | TRUE |  |  |
| updated_at | 更新时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| updated_by | 更新者ID | VARCHAR | 36 | TRUE |  |  |
| is_deleted | 逻辑删除标志 | BOOLEAN |  | TRUE | false |  |

- Table Name: users
- PK: users_pk
