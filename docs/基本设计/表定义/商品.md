# 商品分类主数据表设计

## 商品主数据

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 商品唯一标识 | VARCHAR | 36 | TRUE |  | PK |
| name | 商品名称 | VARCHAR | 200 | TRUE |  |  |
| description | 商品描述 | VARCHAR | 2000 | FALSE |  |  |
| price | 商品价格 | NUMBER(12,2) |  | TRUE |  |  |
| stock | 库存数量 | NUMBER(10) |  | TRUE | 0 |  |
| category_id | 分类ID | VARCHAR | 36 | FALSE |  | FK -> categories.id |
| category | 商品分类 | VARCHAR | 100 | FALSE |  | 分类名称（冗余） |
| image_url | 商品图片URL | VARCHAR | 500 | FALSE |  |  |
| status | 商品状态 | VARCHAR | 20 | TRUE | 'active' | active/inactive |
| created_at | 创建时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| created_by | 创建者ID | VARCHAR | 36 | TRUE |  |  |
| updated_at | 更新时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| updated_by | 更新者ID | VARCHAR | 36 | TRUE |  |  |
| is_deleted | 逻辑删除标志 | NUMBER(1) |  | TRUE | 0 |  |

- Table Name: products
- PK: products_pk
- INDEX: products_category_id_idx (category_id)
- INDEX: products_status_idx (status)
