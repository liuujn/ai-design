# 地址主数据表设计

## 地址主数据

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 地址唯一标识 | VARCHAR | 36 | TRUE |  | PK |
| user_id | 所属用户ID | VARCHAR | 36 | TRUE |  | FK → users.id |
| address_type | 地址类型 | VARCHAR | 20 | TRUE | 'shipping' | shipping/billing |
| recipient_name | 收件人姓名 | VARCHAR | 100 | TRUE |  |  |
| recipient_phone | 收件人电话 | VARCHAR | 20 | TRUE |  |  |
| country | 国家 | VARCHAR | 100 | TRUE | '中国' |  |
| province | 省/州 | VARCHAR | 100 | TRUE |  |  |
| city | 城市 | VARCHAR | 100 | TRUE |  |  |
| district | 区/县 | VARCHAR | 100 | FALSE |  |  |
| street | 详细地址 | VARCHAR | 200 | TRUE |  | 街道+门牌号+详细地址 |
| postal_code | 邮政编码 | VARCHAR | 20 | FALSE |  |  |
| is_default | 是否默认地址 | BOOLEAN |  | TRUE | false | 每个用户仅一个默认地址 |
| created_at | 创建时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| created_by | 创建者ID | VARCHAR | 36 | TRUE |  |  |
| updated_at | 更新时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| updated_by | 更新者ID | VARCHAR | 36 | TRUE |  |  |
| is_deleted | 逻辑删除标志 | BOOLEAN |  | TRUE | false |  |

- Table Name: addresses
- PK: addresses_pk
- FK: addresses_user_fk (user_id → users.id)
- INDEX: addresses_user_idx (user_id)
