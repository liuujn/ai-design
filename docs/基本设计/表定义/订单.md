# 订单主数据表设计

## 订单主表

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 订单唯一标识 | VARCHAR | 36 | TRUE |  | PK |
| order_no | 订单编号 | VARCHAR | 50 | TRUE |  | UNIQUE |
| customer_name | 客户姓名 | VARCHAR | 100 | TRUE |  |  |
| total_amount | 订单总金额 | DECIMAL(12,2) |  | TRUE | 0 |  |
| status | 订单状态 | VARCHAR | 20 | TRUE | 'pending' | pending/confirmed/shipped/completed/cancelled |
| remark | 备注 | VARCHAR | 500 | FALSE |  |  |
| created_at | 创建时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| created_by | 创建者ID | VARCHAR | 36 | TRUE |  |  |
| updated_at | 更新时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| updated_by | 更新者ID | VARCHAR | 36 | TRUE |  |  |
| is_deleted | 逻辑删除标志 | BOOLEAN |  | TRUE | false |  |

- Table Name: orders
- PK: orders_pk

## 订单明细表

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 明细唯一标识 | VARCHAR | 36 | TRUE |  | PK |
| order_id | 订单ID | VARCHAR | 36 | TRUE |  | FK → orders.id |
| item_name | 商品名称 | VARCHAR | 200 | TRUE |  |  |
| quantity | 数量 | INTEGER |  | TRUE | 1 |  |
| unit_price | 单价 | DECIMAL(10,2) |  | TRUE | 0 |  |
| subtotal | 小计金额 | DECIMAL(12,2) |  | TRUE | 0 | quantity * unit_price |
| created_at | 创建时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| updated_at | 更新时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| is_deleted | 逻辑删除标志 | BOOLEAN |  | TRUE | false |  |

- Table Name: order_items
- PK: order_items_pk
- FK: order_items_order_fk (order_id → orders.id)
