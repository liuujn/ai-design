# 购物车主数据表设计

## 购物车主表

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 购物车唯一标识 | VARCHAR | 36 | TRUE |  | PK |
| user_id | 所属用户ID | VARCHAR | 36 | TRUE |  | FK → users.id |
| address_id | 配送地址ID | VARCHAR | 36 | FALSE |  | FK → addresses.id（结算时选择） |
| status | 购物车状态 | VARCHAR | 20 | TRUE | 'active' | active/checked_out |
| total_amount | 购物车总金额 | NUMBER(12,2) |  | TRUE | 0 | 自动计算 |
| remark | 备注 | VARCHAR | 500 | FALSE |  |  |
| created_at | 创建时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| created_by | 创建者ID | VARCHAR | 36 | TRUE |  |  |
| updated_at | 更新时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| updated_by | 更新者ID | VARCHAR | 36 | TRUE |  |  |
| is_deleted | 逻辑删除标志 | NUMBER(1) |  | TRUE | 0 |  |

- Table Name: carts
- PK: carts_pk
- FK: carts_user_fk (user_id → users.id)
- FK: carts_address_fk (address_id → addresses.id)
- INDEX: carts_user_idx (user_id)
- INDEX: carts_status_idx (status)

## 购物车明细表

|項目ID|項目名|型|長さ|NOT NULL|デフォルト値|備考|
|---|---|---|---|---|---|---|
| id | 明细唯一标识 | VARCHAR | 36 | TRUE |  | PK |
| cart_id | 购物车ID | VARCHAR | 36 | TRUE |  | FK → carts.id |
| product_id | 商品ID | VARCHAR | 36 | TRUE |  | FK → products.id |
| product_name | 商品名称 | VARCHAR | 200 | TRUE |  | 冗余（商品改名后保留快照） |
| quantity | 数量 | NUMBER(10) |  | TRUE | 1 |  |
| unit_price | 单价 | NUMBER(12,2) |  | TRUE | 0 | 加入购物车时的价格 |
| created_at | 创建时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| created_by | 创建者ID | VARCHAR | 36 | TRUE |  |  |
| updated_at | 更新时间 | TIMESTAMP |  | TRUE | CURRENT_TIMESTAMP |  |
| updated_by | 更新者ID | VARCHAR | 36 | TRUE |  |  |
| is_deleted | 逻辑删除标志 | NUMBER(1) |  | TRUE | 0 |  |

- Table Name: cart_items
- PK: cart_items_pk
- FK: cart_items_cart_fk (cart_id → carts.id)
- FK: cart_items_product_fk (product_id → products.id)
- INDEX: cart_items_cart_idx (cart_id)
